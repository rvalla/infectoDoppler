iD: player guide

simulation

The players are guided by the animation while making their sounds. They have to modulate these sounds applying to them the Doppler effect induced by the motion of a selected asteroid and a star. The animation is based on the gravitational model from The coding train. It is very real for asteoids but much less gravitational for stars.

input level

After the users give permission to access the microphone or their devices playerguide.js allows them to adjust the input level. In case de mic=false parameter is passed, the script jumps this step. It is possible to pass the input amplitude directly too using the micamp=float parameter (with 0 < f <= 1).

animation

The number of stars and asteoroids can be set through url parameters too. You can use sn=n to define number of stars and an=n to define the number of asteroids (n must be an integer). You can control the power of the gravity field passing the gravitational constant with gc=f.
